## Tecnologias Usadas
- Frontend: React + Vite + Typescript
- Backend: FastAPI + Postgres + Redis (Dockerizado)

## Run Locally

**Prerequisites:**  Node.js e Docker

1. Install dependencies no Frontend:
   `npm install`
2. Run the frontend app:
   `npm run dev`
3. Backend:
   `cd backend` e `docker-compose up -d`
